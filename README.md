# Temperature Anomaly Analysis (2015-2019)

## 📌 Project Overview
This project analyzes global temperature anomalies from 2015 to 2019 to identify trends in climate change.

## 🔍 Data Source
- The dataset contains temperature anomalies in °C based on climatology (1901-2000).

## 📊 Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📈 Key Findings
- 2016 recorded the highest temperature anomaly (1.00°C).
- The lowest anomaly was in 2018 (0.83°C).
- The overall trend suggests a rise in temperature anomalies.

## 🚀 How to Use
1. Clone this repository: git clone https://github.com/Shrutax/Temperature_Anomaly_Analysis.git
2. Open the Jupyter Notebook and run the code.
